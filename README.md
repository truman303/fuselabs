# FuseLab Digital Website

This is the repository for the FuseLab Digital website.

## Quick Start

npm run dev       # dev server
npm run build     # static build → dist/
npm run preview   # preview the build
npm run check     # astro type check

## Tech Stack

- Astro
- GSAP
- Cloudflare Pages
